$|x - 2| + |x + 2| = 8$
1. $x \geq 2$
$x - 2 + x + 2 = 8$
$2x = 8$
$x = 4$
2. $-2 \leq x < 2$
$-x + 2 + x + 2 = 8$
$4 \neq 8$
3. $x < -2$
$-x + 2 - x - 2 = 8$
$-2x = 8$
$x = -4$
